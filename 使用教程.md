# api聚合反代 使用教程

## 1. 安装与启动

1. 下载 `安装包/api聚合反代-WIN-X64-1.0.24.exe`。
2. 双击安装包，根据安装向导完成安装。
3. 安装完成后打开 `api聚合反代`。
4. 应用启动后会在本机提供管理界面和 OpenAI 兼容接口。

默认管理界面：

```text
http://127.0.0.1:23456
```

默认 API 地址：

```text
http://127.0.0.1:23456/v1
```

## 2. 添加 API

可以手动添加，也可以在「模板导入」中下载 Excel 模板后批量导入。建议把同一模型或同一用途的接口放在同一分组。

## 3. 配置分组调用

分组名称就是外部工具调用时的模型 ID。例如分组名为 `gpt-5.5`，外部工具里就把 Model 填成 `gpt-5.5`。

在「API 管理」中展开分组后，可以查看每条 API 的请求次数、Tokens、输入/输出消耗、价格、状态和操作按钮。如果窗口较窄，在表格内横向滚动即可看到全部列。

## 4. 令牌与外部调用

在「令牌管理」中创建令牌，然后在外部工具中填写：

```text
Base URL: http://127.0.0.1:23456/v1
API Key: 你在令牌管理中创建的令牌
Model: API 管理中的分组名称
```

## 5. 查看用量

「数据看板」和「使用日志」可以查看请求量、Tokens、成功/失败状态和消耗金额。如果调用失败，使用日志会帮助你定位错误原因。
