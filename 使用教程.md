# 使用教程

## 1. 安装并启动

1. 下载最新版 `api聚合反代-WIN-X64-版本号.exe` 安装包。
2. 按安装向导完成安装。
3. 启动应用后，浏览器管理界面会显示本机服务地址。

默认管理界面：`http://127.0.0.1:23456/`

默认 API 调用地址：`http://127.0.0.1:23456/v1`

## 2. 导入 API

1. 打开左侧菜单中的“模板导入”。
2. 下载模板，按模板填写 API 地址、Key、模型、分组和价格信息。
3. 上传 Excel 文件，系统会按分组导入或更新 API。
4. 分组名称就是模型 ID，外部工具里复制分组名称填写模型 ID 即可。

## 3. 创建令牌并调用

1. 打开“令牌管理”。
2. 创建一个调用令牌。
3. 在 OpenAI 兼容客户端中配置：
   - Base URL：`http://127.0.0.1:23456/v1`
   - API Key：复制应用中的令牌
   - Model：填写 API 管理里的分组名称

## 4. 查看用量

- “数据看板”可以查看总体请求、Tokens、费用和模型分析。
- “API 管理”展开分组后，可以查看每个 API 的请求、Tokens、费用、价格和状态。
- “使用日志”可以查看每次调用记录、状态码、耗时和消耗明细。

## 5. 更新

下载安装新版安装包后覆盖安装即可。历史本地数据会保留在用户数据目录中，安装包不会携带调试电脑上的 API 数据或登录状态。
