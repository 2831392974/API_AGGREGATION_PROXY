# 使用教程

## 安装与启动

1. 在 Releases 页面或 `安装包` 文件夹下载最新版 Windows 安装包。
2. 双击安装包并按提示完成安装。
3. 启动应用后，浏览器控制台会打开本地管理界面。
4. 如浏览器没有自动打开，可手动访问 `http://127.0.0.1:23456/`。

## 基础配置

1. 打开“模板导入”，下载 Excel 模板。
2. 在模板中填写 API 地址、Key、模型 ID、价格等信息。
3. 上传模板后，在“API 管理”中查看分组和接口状态。
4. 在“令牌管理”中创建调用令牌。
5. 外部工具中填写：
   - Base URL：`http://127.0.0.1:23456/v1`
   - API Key：令牌管理中创建的令牌
   - 模型 ID：API 管理中的分组名称

## 常用功能

- API 管理：查看分组、接口状态、请求次数、Tokens 和消耗明细。
- 数据看板：查看整体调用趋势和模型数据分析。
- 使用日志：排查请求成功、失败状态和错误代码。
- 测试对话：直接测试已导入的模型分组。
- 白嫖分享：查看可用网站分享内容。
- 用户管理：管理员可以控制新用户注册、查看注册申请、编辑用户状态并按用户类型批量发送通知邮件。

## 数据与隐私

本工具不会把公开仓库作为用户数据存储位置。安装包不包含源代码、本地数据库、`.env`、日志、登录状态或用户导入的 API 数据。用户的 API Key 和接口配置默认保留在本机运行数据目录中。
