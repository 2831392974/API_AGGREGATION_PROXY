# 使用教程

## 安装与启动

1. 打开 `安装包` 文件夹，下载最新的 `api聚合反代-WIN-X64-版本号.exe`。
2. 双击安装包，根据向导完成安装。
3. 启动应用后，默认会在本机打开管理控制台。
4. 如需外部工具调用，请在应用中创建令牌，然后使用本机 `/v1` 兼容接口地址和令牌。

## API 管理

1. 在 API 管理页面创建分组。
2. 手动添加 API，或使用 Excel 模板批量导入。
3. 价格字段支持输入、输出、缓存读取、缓存创建、补全和按请求等计费类型。
4. 应用会按分组策略进行轮换，并记录调用日志和用量统计。

## 应用管理

应用管理页面会检测常见 Agent Code / 桌面 Agent 应用。已安装应用会排在前面并可启动，未安装应用会置灰。启动时只注入本项目的 API 代理环境变量，不会修改目标应用原来的记忆、技能、设置或用户数据目录。

## 隐私与安全

公共安装包不包含 `.env`、数据库连接串、邮箱授权码、Supabase secret 或用户本地 API 数据。用户导入的 API Key、分组、Token、日志和模型用量默认只保存在本机。`1.0.36` 起公共联机能力通过 Cloudflare Worker + D1 提供，客户端只知道 Worker 公共入口。

## Docker/NAS

Docker 子项目位于主项目中的 `docker部署api` 文件夹。服务器部署时请自行配置端口、数据卷和必要环境变量；默认不会把本机私有 `.env` 打包到公共安装包中。
