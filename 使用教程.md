# 使用教程

## 1. 安装启动

1. 在 Releases 页面下载最新版本安装包。
2. 双击安装包，按安装向导完成安装。
3. 启动应用后注册或登录账号。
4. 进入控制台后，可在顶部看到本机服务地址。

## 2. 配置 API

1. 打开“模板导入”页面，下载 API 导入模板。
2. 按模板填写分组名称、接口地址、API Key、模型名称和价格信息。
3. 导入后进入“API 管理”页面查看分组。
4. 分组名称就是模型 ID，复制分组名称填写到 OpenAI 兼容客户端即可。

## 3. 调用地址

默认本机调用地址：

```text
http://127.0.0.1:23456/v1
```

如果部署在 NAS 或服务器上，请使用对应设备的访问地址并追加 `/v1`。

## 4. 令牌管理

在“令牌管理”页面创建自己的调用令牌。令牌按账户绑定，登录同一账号后可同步自己的令牌数据。

## 5. 数据看板与日志

- “数据看板”用于查看请求量、Token、消耗金额和模型分析。
- “使用日志”用于查看每次调用记录、状态和错误代码。
- “测试对话”可直接测试自己配置的分组和模型。

## 6. 注意事项

- 请不要把自己的 API Key 分享给他人。
- 不同用户之间的 API、令牌、日志和策略互相隔离。
- 如果在公司电脑、NAS 或服务器部署，请确认端口没有被占用。
