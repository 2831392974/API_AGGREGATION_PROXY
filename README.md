# API 聚合反代

API 聚合反代是一款面向本地电脑、局域网、NAS 和服务器部署的 API 聚合、反向代理和调用管理工具。它提供 OpenAI 兼容接口地址，方便把多个模型分组统一接入到支持 OpenAI API 的客户端、自动化工具或业务脚本中。

## 主要能力

- 按分组统一管理多个 API，分组名称可直接作为模型 ID 使用。
- 支持查看分组请求量、Token 用量、消耗金额、调用状态和价格明细。
- 支持通过 Excel 模板批量导入 API 配置，适合维护多模型、多供应商配置。
- 支持令牌管理、使用日志、数据看板和测试对话，便于排查调用效果。
- 支持文本、图像相关接口的统一管理与测试。
- 支持 Windows 安装包使用，也提供 Docker/NAS 部署方案用于局域网或服务器环境。

用户导入的 API Key、接口地址和策略数据按账户隔离处理，不会上传到本公开仓库。
