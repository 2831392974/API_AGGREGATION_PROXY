# API聚合反代

API聚合反代是一款面向本地和局域网部署的 API 聚合、反向代理和调用管理工具。它提供 OpenAI 兼容接口地址，方便把多个模型分组统一接入到支持 OpenAI API 的客户端或自动化工具中。

## 主要能力

- 统一管理多个 API 分组，分组名称可直接作为模型 ID 使用。
- 支持按分组查看请求次数、Token 用量、消耗金额、状态和价格明细。
- 支持模板导入 API 配置，便于批量维护模型和供应商配置。
- 支持令牌管理、调用日志、数据看板和测试对话，方便排查调用效果。
- 支持白嫖分享网站内容联机同步，管理员审核后可展示给用户。
- 支持账号登录和跨设备同步必要账户数据，用户自己的 API、令牌、策略和日志按账号隔离。
- 支持 Windows 安装包使用，也提供 Docker/NAS 部署方案用于局域网或服务器环境。
