# api聚合反代

`api聚合反代` 是一款 Windows 桌面应用，用来把多个 API 服务统一整理成一个本地 OpenAI 兼容接口。安装后即可在本机管理 API、分组、令牌、调用日志和用量统计，适合把不同平台、不同模型、不同接口地址统一接入到同一个调用入口。

## 下载安装

最新版安装包：

[安装包/api聚合反代-WIN-X64-1.0.18.exe](安装包/api聚合反代-WIN-X64-1.0.18.exe)

安装完成后打开应用，默认本地服务地址为：

```text
http://127.0.0.1:23456
```

OpenAI 兼容接口地址为：

```text
http://127.0.0.1:23456/v1
```

## 主要功能

- API 统一管理：集中维护不同平台的接口地址、密钥、模型名称、价格和启用状态。
- 分组调用：把多个 API 放入同一个分组，外部工具只需要调用分组名称即可自动分发请求。
- Excel 批量导入：支持通过模板批量导入 API，适合一次性整理大量接口配置。
- OpenAI 兼容代理：提供 `/v1/chat/completions`、`/v1/models` 等常用兼容接口，方便接入支持 OpenAI 格式的软件。
- 图像接口支持：支持文本模型和图像模型分开测试与调用，适配文生图、图生图等接口场景。
- 调用日志统计：记录请求量、tokens、消耗、成功率和失败原因，方便定位接口质量和成本变化。
- 本地隐私保护：用户导入的 API Key、接口地址和分组策略默认保存在本机，不作为公开文件上传。

## 使用优势

- 安装即用，不需要手动部署后端服务。
- 一个入口管理多个 API，减少不同软件里重复填写接口配置的麻烦。
- 支持拖拽调整 API 优先级，便于把稳定或低成本接口放在前面。
- 支持复制接口地址、令牌和模型名称，方便快速接入第三方工具。
- 数据看板可以直观看到调用趋势、接口消耗和模型使用情况。
- 配置和日志在本地留存，方便个人或团队内部排查问题。

## 快速开始

1. 下载并运行 `api聚合反代-WIN-X64-1.0.18.exe`。
2. 启动应用后进入控制台。
3. 在 `API管理` 中手动添加 API，或在 `模板导入` 中下载模板后批量导入。
4. 在 `分组管理` 中确认分组名称，外部软件调用时把分组名称作为模型 ID。
5. 在 `令牌管理` 中复制可用令牌。
6. 在需要接入的工具中填写：

```text
Base URL: http://127.0.0.1:23456/v1
API Key: 令牌管理中复制的令牌
Model: 分组管理中的分组名称
```

## 隐私说明

本项目只记录应用运行所需的账户数据和本地调用数据。用户导入的 API Key、接口地址、分组策略等隐私配置默认保存在本机文件中，请不要把本机数据库文件或 `.env` 文件上传到公开仓库。
