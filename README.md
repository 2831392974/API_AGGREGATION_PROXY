# API 聚合反代

API 聚合反代是一款 Windows 桌面工具，用于把多个 AI API 接口统一整理到本地控制台中，方便进行分组管理、统一转发、状态检测、令牌调用和用量统计。

## 主要用途

- 统一管理不同来源的 API 地址、Key、模型名称和价格配置。
- 按分组生成模型 ID，外部工具只需要填写统一的 `/v1` 地址和令牌即可调用。
- 支持 Excel 模板批量导入，适合一次性整理大量接口。
- 支持接口健康测试、自动检测、使用日志、请求次数、Tokens 和消耗金额统计。
- 支持本地浏览器控制台和桌面应用方式使用，安装后即可启动服务。
- 支持网站分享、聊天测试、数据看板等辅助功能。

## 隐私说明

用户导入的 API Key、接口地址、分组策略等配置默认保存在本机数据目录中。公开仓库只发布安装包、说明文档和更新说明，不包含源代码、`.env`、数据库、日志、登录状态或任何本地 API 数据。
