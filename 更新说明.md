# 1.0.25 更新说明

发布日期: 2026-07-09

本次公开发布内容如下:

1. 部署并启用用户工作区云端同步表和安全 RPC，支持用户 API 分组、API、日志、令牌和策略按账号隔离同步。
2. 修复 Docker/NAS 部署副本缺少用户工作区同步逻辑的问题，避免不同账号之间 API 数据串用。
3. 修复令牌列表用量统计可能跨用户统计日志的问题。
4. 修复 Windows 安装过程中的许可协议中文乱码问题。
5. 同步更新 Docker 部署副本到 1.0.25，继续保留 Docker 默认端口 3039 和 Cloudflare 快速隧道配置。

## 下载

最新版本安装包:
