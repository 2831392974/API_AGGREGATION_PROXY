# 更新说明

## 1.0.31

发布日期：2026-07-12

- 增强“应用管理”页面，新增应用模板入口，支持快速添加 Visual Studio Code、Cursor、Windsurf、Trae、Web 控制台和 OpenAI 兼容接口。
- 系统应用启动时只向本次子进程注入本项目代理环境变量，不修改系统全局配置、会话记录、用户数据目录或技能仓库。
- Web 应用启动改为返回 URL 给前端，由当前用户浏览器打开；浏览器模式会在点击瞬间打开占位页，接口返回后跳转，降低 NAS/Web 模式弹窗拦截概率。
- Docker/NAS 副本同步同一套应用管理能力，继续固定 `3039` 端口，保留 `cloudflared` 快速隧道，并默认禁用宿主机系统应用启动。
- 本次只发布 Windows 主项目安装包，Docker/NAS 副本仍作为独立部署项目隔离维护，不打进桌面安装包。

## 1.0.30

发布日期：2026-07-10

- 新增“应用管理”页面，支持按账号维护常用应用列表、应用启动配置、代理注入设置和启动日志。
- 支持自动检测并添加 Web 控制台和 Visual Studio Code；VS Code 启动时可注入本项目代理地址环境变量。
- 支持为应用选择当前用户令牌并通过 `API_AGGREGATOR_API_KEY` / `OPENAI_API_KEY` 注入，不修改系统全局配置、会话记录或技能仓库。
- 应用配置和启动日志纳入用户工作区，不同用户互不干扰。
- Docker/NAS 副本同步应用管理页面和配置能力，并默认禁用宿主机系统应用启动，避免多用户服务器环境误启动宿主机桌面应用。

## 1.0.29

发布日期：2026-07-10

- 修复浏览器、NAS、服务器和 Cloudflare 隧道网页模式下白嫖分享链接可能无法打开的问题。
- 白嫖分享链接点击时会先在用户浏览器中占位打开，拿到真实链接后跳转；如果新窗口被拦截，会回退到当前标签页打开。
- 扩展用户管理页面批量邮件目标，支持全部正常、7 天内到期、已到期、已停用、已封禁、注册申请邮箱和当前勾选用户。
- 注册申请窗口新增“给申请邮箱发邮件”入口，方便管理员通知申请人。
- 复核新用户默认无令牌、令牌按账号绑定、不同用户分组和工作区快照互不混入。
- 同步 Docker/NAS 部署副本，继续保持 Docker 默认端口 3039 与 Cloudflare 快速隧道配置。

## 1.0.28

发布日期：2026-07-10

- 完成联机同步验收：套餐、用户、兑换码、白嫖分享可正常云端读取，管理员云端写入不依赖分发安装包携带 service-role 密钥。
- 完成用户隔离验收：不同用户的 API 分组、API、令牌、策略、日志、统计和工作区快照均按账号隔离。
- 验证 `/v1/models` 令牌鉴权：无令牌返回 401，有效用户只能看到自己的模型，过期账号返回 402。
- 同步可选 Cloudflare Worker 后端的套餐软删除逻辑，避免删除过的套餐在远端列表中重新出现。
- 复核安装器中文协议资源为 UTF-8 BOM 编码，继续降低安装界面中文乱码风险。
- 安装包继续排除 `.env`、数据库、本地 Excel/API 数据、登录状态、日志和调试配置。
