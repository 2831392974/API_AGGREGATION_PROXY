# 1.0.21 更新说明

发布日期: 2026-07-04

本次公开发布内容如下:

1. 优化 API 管理页中分组展开后的单条 API 横向表格展示。
2. 将 API 名称和模型合并为 `API / 模型` 主列，减少横向列数，提升窗口内可视空间。

5. 刷新前端资源版本，避免更新后仍读取旧样式缓存。

# 1.0.20 更新说明

发布日期: 2026-07-03

本次公开发布内容如下:

1. 白嫖分享页面的网站链接改为使用系统默认浏览器打开，点击后不再在应用内部新开窗口。
2. 普通用户默认只看到 `访问链接` 按钮，完整链接仅对发布者本人可见，页面展示更简洁。
3. 分享网站新增标签能力，提交和展示时都可以通过标签标记网站用途或类型。
4. 新用户默认体验周期调整为 365 天，便于安装后直接完成长期测试和使用。
5. 优化云端同步字段，确保网站分享内容、标签和用户体验周期可以跨设备同步。
6. 刷新前端资源版本，避免安装后继续读取旧缓存导致页面显示不是最新效果。

# 1.0.19 更新说明

发布日期: 2026-07-02

本次公开发布内容如下:

1. 分组管理页支持展开分组后拖拽调整 API 优先级。
2. 分组管理页新增分组请求次数、Tokens、消耗展示，展开后也能查看每个 API 的用量数据。
3. 分组勾选框移动到分组名称前方，支持批量测试和批量删除。
4. API 管理页优化表格宽度，新增批量测试 API，保留请求、Tokens、消耗排序。
5. 优化 Excel 批量导入，修复隐藏超链接地址和空单元格导致漏导入的问题。
6. 优化“单独模型”导入规则，避免把该页面名称重复导入为普通分组。
7. 优化数据看板布局，提升窗口尺寸变化时的展示稳定性。
8. 调整顶部复制 API 地址按钮和用户名展示位置。
