# 更新说明

## 1.0.27

发布日期：2026-07-09

- 优化 API 管理页面中分组展开后的横向表格布局，窗口较窄时可以在表格内部横向滚动查看完整信息。
- 单个 API 的消耗金额展示补充输入、输出费用明细，样式与 Tokens 明细保持一致。
- 优化单个 API 操作按钮的横向排列，减少窄窗口下按钮竖排和信息挤压。
- 刷新前端资源缓存版本，避免升级后继续读取旧版样式。
- 调整安装器许可证协议和自定义说明文件的编码生成方式，继续修复安装过程中文显示乱码风险。
- 同步 Docker/NAS 部署副本，继续保持 Docker 默认端口 3039 与 Cloudflare 快速隧道配置。

## 1.0.26

- 优化 Excel 批量导入逻辑，从删除重建改为差异更新，尽量保留已有 API ID、日志和数据看板统计。
- 重复导入同一文件时会更新已有 API，不会重复新增。
- 替换导入时只移除当前用户文件中确实不存在的 API 或空分组，不清理历史日志和统计。

## 1.0.25

- 增强云端用户工作区同步，按账号隔离 API 分组、API、日志、令牌和策略。
- 修复 NAS/Docker 环境下不同用户数据互相读取或调用的风险。
- 优化 Windows 安装器中文协议生成方式。

## 1.0.20 - 1.0.24

- 优化网站分享、数据看板、API 管理、日志展示和导入模板体验。
- 修复部分浏览器、NAS 和 Cloudflare 快速隧道环境下的链接与端口适配问题。
- 持续排除 `.env`、数据库、本地 Excel/API 数据、登录状态、日志和调试配置。
