# 1.0.18 更新说明

发布日期: 2026-07-01

本次公开发布内容如下:

1. 提供 Windows x64 安装包，安装完成后即可在本机启动 `api聚合反代`。
2. 修复安装器欢迎页、许可证协议页和安装过程说明文字的中文乱码问题。
3. 优化 Excel 批量导入逻辑，提升模板导入时的识别稳定性。
4. 优化 API 管理表格交互，支持按请求量、tokens 和消耗查看接口使用情况。
5. 优化图像类 API 的测试与调用路径，便于区分文本接口和图像接口。
6. 优化复制操作，确保接口地址、令牌和模型名称复制后能正确写入系统剪贴板。
7. 优化数据看板和页面布局，提升窗口尺寸变化时的展示稳定性。

## 下载

最新版安装包：

```text
安装包/api聚合反代-WIN-X64-1.0.18.exe
```
