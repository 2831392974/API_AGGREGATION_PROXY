# 更新说明

## 1.0.32

发布日期：2026-07-12

- 修复聊天窗口在历史消息或图片字段为空时可能出现 `Cannot read properties of null (reading 'image')` 的崩溃问题，并对本地对话记忆做兼容清洗。
- 优化聊天页面布局，左侧模型和配置选择栏保持固定，页面滚动只影响右侧对话内容；同时修正重新发送、重新生成时复用错误 payload 的问题。
- 重做应用管理页面为应用启动器式布局，默认内置主流 Agent Code / 桌面 Agent 应用目录，支持自动检测、图标展示、已安装点亮、未安装置灰禁用和启动分组选择。
- 扩展应用目录，覆盖 Codex、Claude Desktop、Claude Code、Gemini Desktop、Gemini CLI、OpenCode Desktop、OpenCode、Trae、TRAE SOLO CN、Trae CN、Cursor、VS Code、Windsurf、Zed、ZCode、WorkBuddy、Hermes Desktop、Grok Build、Kiro、Void 等常见应用。
- 优化白嫖分享页面通知，改为右上角浮窗式消息栈，进入页面后自动展示并在数秒后收起，不再作为常驻面板占用页面空间。
- 同步更新 Docker/NAS 部署子项目，保持桌面主项目与 docker部署api 子项目功能一致。
