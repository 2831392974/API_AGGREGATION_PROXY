# 更新说明

## 1.0.29

发布日期：2026-07-10

- 修复浏览器、NAS、服务器和 Cloudflare 隧道网页模式下白嫖分享链接可能无法打开的问题。
- 白嫖分享链接点击时会先在用户浏览器中占位打开，拿到真实链接后跳转；如果新窗口被拦截，会回退到当前标签页打开。
- 扩展用户管理页面批量邮件目标，支持全部正常、7 天内到期、已到期、已停用、已封禁、注册申请邮箱和当前勾选用户。
- 注册申请窗口新增“给申请邮箱发邮件”入口，方便管理员通知申请人。
- 复核新用户默认无令牌、令牌按账号绑定、不同用户分组和工作区快照互不混入。
- 同步 Docker/NAS 部署副本，继续保持 Docker 默认端口 3039 与 Cloudflare 快速隧道配置。

## 1.0.28

发布日期：2026-07-10

- 完成联机同步验收：套餐、用户、兑换码、白嫖分享可正常云端读取，管理员云端写入不依赖分发安装包携带 service-role 密钥。
- 完成用户隔离验收：不同用户的 API 分组、API、令牌、策略、日志、统计和工作区快照均按账号隔离。
- 验证 `/v1/models` 令牌鉴权：无令牌返回 401，有效用户只能看到自己的模型，过期账号返回 402。
- 同步可选 Cloudflare Worker 后端的套餐软删除逻辑，避免删除过的套餐在远端列表中重新出现。
- 复核安装器中文协议资源为 UTF-8 BOM 编码，继续降低安装界面中文乱码风险。
- 安装包继续排除 `.env`、数据库、本地 Excel/API 数据、登录状态、日志和调试配置。

## 1.0.27

- 优化 API 管理页面中分组展开后的横向表格布局，窗口较窄时可以在表格内部横向滚动查看完整信息。
- 单个 API 的消耗金额展示补充输入、输出费用明细，样式与 Tokens 明细保持一致。
- 优化单个 API 操作按钮的横向排列，减少窄窗口下按钮竖排和信息挤压。
- 刷新前端资源缓存版本，避免升级后继续读取旧版样式。
- 调整安装器许可证协议和自定义说明文件的编码生成方式，继续修复安装过程中文显示乱码风险。
- 同步 Docker/NAS 部署副本，继续保持 Docker 默认端口 3039 与 Cloudflare 快速隧道配置。

## 1.0.26

- 优化 Excel 批量导入逻辑，从删除重建改为差异更新，尽量保留已有 API ID、日志和数据看板统计。
- 重复导入同一文件时会更新已有 API，不会重复新增。
- 替换导入时只移除当前用户文件中确实不存在的 API 或空分组，不清理历史日志和统计。

## 1.0.25

- 增强云端用户工作区同步，按账号隔离 API 分组、API、日志、令牌和策略。
- 修复 NAS/Docker 环境下不同用户数据互相读取或调用的风险。
- 优化 Windows 安装器中文协议生成方式。

## 1.0.20 - 1.0.24

- 优化网站分享、数据看板、API 管理、日志展示和导入模板体验。
- 修复部分浏览器、NAS 和 Cloudflare 快速隧道环境下的链接与端口适配问题。
- 持续排除 `.env`、数据库、本地 Excel/API 数据、登录状态、日志和调试配置。
