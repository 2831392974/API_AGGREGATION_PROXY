# 1.0.26 更新说明

发布日期：2026-07-09

本次公开发布内容如下：

1. 优化 Excel 批量导入逻辑，从“删除重建”改为差异更新，尽量保留原有 API ID、创建时间、健康状态和历史统计关联。
2. 重复导入同一模板时会更新已有 API，而不是重复新增，减少大批量导入后的重复数据。
3. 替换模式下只移除当前用户在新模板中已经不存在的 API 和空分组，不清理使用日志和数据看板历史统计。
4. 优化导入性能，减少逐行保存和云端同步开销，更适合大文件导入。
5. 同步 Docker/NAS 部署副本，保持导入逻辑和 Windows 版本一致。

## 下载
